# Bowling

A lightweight HTML5 bowling game with:

- 10 standard frames with strike, spare, and final-frame bonus scoring
- left/right lane positioning before each roll
- a quick-action curve meter that lets you hook the ball mid-roll
- exploding pin effects when pins are knocked down

## Run locally

Open `/home/runner/work/Bowling/Bowling/index.html` in a browser, or serve the repo root with a simple static server such as:

```bash
cd /home/runner/work/Bowling/Bowling
python3 -m http.server 8000
```
